# Currículo em LaTeX 📄

Este repositório contém o código-fonte em LaTeX do meu currículo profissional. O uso do LaTeX garante uma formatação consistente, profissional e facilita o controle de versão das minhas atualizações de carreira.

## 📂 Estrutura do Projeto

O arquivo principal `Curriculo-.tex` contém as seguintes seções estruturadas:
*   **Informações de Contato** (Links profissionais, e-mail, telefone)
*   **Resumo Profissional**
*   **Experiência Profissional**
*   **Habilidades Técnicas**
*   **Projetos em Destaque**
*   **Formação Acadêmica**

## 🚀 Como Visualizar e Gerar o PDF

Para converter o arquivo `.tex` em um documento PDF, você pode usar uma das opções abaixo:

### Opção 1: Usando o Overleaf (Mais simples, sem instalar nada)
1. Acesse o [Overleaf](https://www.overleaf.com/) (editor LaTeX online gratuito).
2. Crie um novo projeto vazio.
3. Copie e cole o conteúdo de `Curriculo-.tex` no editor.
4. Clique no botão **Recompile** para gerar e baixar o PDF finalizado.

### Opção 2: Compilação Local
Caso prefira compilar no seu computador:
1. Instale uma distribuição LaTeX (como o **TeX Live** para Windows/Linux ou **MacTeX** para macOS).
2. Use um editor de texto de sua preferência (como o **VS Code** com a extensão *LaTeX Workshop*).
3. Compile o arquivo `Curriculo-.tex` para gerar o arquivo `.pdf`.
